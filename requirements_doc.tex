\section*{System requirements}

\begin{itemize}
    \item \textbf{Operating system}: Linux, macOS, or Windows.
    \item \textbf{Python}: 3.8 or later (tested on 3.10 and 3.11).
    \item \textbf{Disk}: approximately 50\,MB for datasets and results.
    \item \textbf{RAM}: 1\,GB minimum (2\,GB recommended for 30-trial runs).
\end{itemize}

\section*{Python packages}

All packages are listed in \texttt{requirements.txt} and can be installed with:

\begin{verbatim}
pip install -r requirements.txt
\end{verbatim}
